## When To Create A New Vertical File

Create a new file when:
1. You've researched the vertical for at least one client
2. The research contains insights that would apply to 2+ future clients in
   the same space

Filename slugs (examples):
- `podcast-production.md`
- `fitness-coaching.md`
- `ecommerce-skincare.md`
- `b2b-saas.md`
- `wedding-photography.md`
